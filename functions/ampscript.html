Categorized reference of common AMPscript functions used in Salesforce Marketing Cloud.

📝 String Functions

Concatenation, length, case conversion, and more.

📆 Date Functions

Date formatting, offset, and comparisons.

Lookup, retrieve, update, and insert data.

Send email, personalization strings, subscriber context.

Conditionals, comparisons, and branching.

🔧 Other Functions

Base64, encryption, error handling, and more.

📝 String Functions

  • Concat() – Concatenates multiple strings.
  • Length() – Returns the length of a string.
  • Lowercase() – Converts text to lowercase.
  • Uppercase() – Converts text to uppercase.
  • ProperCase() – Converts text to proper case.

📆 Date Functions

  • Now() – Returns the current system date and time.
  • FormatDate() – Formats a date using a given format string.
  • DateAdd() – Adds an interval to a date.
  • DateDiff() – Returns the difference between two dates.

📂 Data Functions

  • Lookup() – Retrieves a value from a data extension.
  • LookupRows() – Returns multiple rows from a data extension.
  • UpdateDE() – Updates a row in a data extension.
  • InsertDE() – Inserts a new row into a data extension.

✉️ Email Functions

  • AttributeValue() – Returns the value of an attribute.
  • RequestParameter() – Gets values from query strings or form posts.
  • RedirectTo() – Redirects the browser to a new URL.

🔁 Logical Functions

  • IF, ELSEIF, ELSE, ENDIF – Conditional logic.
  • Empty() – Checks if a value is empty.
  • IsEmailAddress() – Validates email address format.

🔧 Other Functions

  • Base64Encode(), Base64Decode()
  • EncryptSymmetric(), DecryptSymmetric()
  • RaiseError() – Throws a custom error in AMPscript.
