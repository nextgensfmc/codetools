✅ AMPscript Reference

Use the following AMPscript functions and examples in emails, CloudPages, and messages in SFMC. Built-in syntax and debug tips included.

Function Description Example Debug Tip
Lookup Retrieve value from a Data Extension 
SET @fname = Lookup("Members", "FirstName", "Email", @email)
 Ensure DE and field names are correct, use double quotes
Concat Combine multiple strings 
SET @name = Concat(@first, " ", @last)
 Use double quotes for strings, not single
FormatDate Convert a date to readable format 
SET @fDate = FormatDate(@today, "MMMM dd, yyyy")
 Check if date value is valid before formatting
Now Returns current system time 
SET @today = Now()
 Use FormatDate() to display output properly
RaiseError Force AMPscript to throw an error 
RaiseError("Missing subscriber key", true)
 Useful for debugging inside IF statements
