✅ Data Views Reference

Built-in tracking data views in SFMC. Use these in SQL Query Activities.

Topic Description Example Debug Tip
_SubscribersAll subscribers
SELECT * FROM _Subscribers
Use SubscriberKey not EmailAddress
_OpenEmail opens
SELECT * FROM _Open WHERE EventDate >= GetDate()
Only 6 months of data retained
_ClickLink click tracking
SELECT * FROM _Click WHERE URL LIKE '%unsubscribe%'
Combine with _Job for context
_SentSent tracking
SELECT * FROM _Sent WHERE JobID = 12345
JobID links to _Job table
_UnsubscribeUnsub events
SELECT * FROM _Unsubscribe
Join with _Subscribers for details
